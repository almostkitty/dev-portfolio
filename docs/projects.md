## 2. Проект ```UBot```

Проект для личного пользования. Telegram-бот для скачивания аудио с YouTube. Конвертирует видео в формат MP3 и отправляет пользователям. Количество зарегистрированных пользователей – 11.

Особенность бота – работа с длинными видео. Также реализовано условное кэширование ранее обработанных фрагментов.

**Ссылка на GitHub:** [https://github.com/almostkitty/UBot](https://github.com/almostkitty/UBot)
